# CS Round Punisher 💪

Простой веб-сайт на HTML, CSS и JavaScript. Платишь телом за каждый проигранный раунд!

## Структура проекта

```
cs-round-punisher-simple/
├── index.html       # Основная страница
├── styles.css       # Стили (CSS)
├── script.js        # Логика (JavaScript)
└── README.md        # Этот файл
```

## Как использовать

1. Откройте `index.html` в браузере
2. Нажмите кнопку "НАЧАТЬ ИГРУ"
3. Позже здесь будет полная игра!

## Технологии

- **HTML5** — структура страницы
- **CSS3** — стили и адаптивность
- **Vanilla JavaScript** — логика без фреймворков

## Палитра цветов

- Фон: `#1A1208` (темный коричнево-черный)
- Текст: `#F0E0C0` (кремово-бежевый)
- Акцент: `#D4952A` (золотой янтарь)
- Мuted: `#A89880` (серо-коричневый)

## Шрифты

- **Libre Baskerville** — заголовки
- **DM Sans** — метки и кнопки
- **Inter** — основной текст

## Адаптивность

Сайт полностью адаптивен:
- ✅ Десктоп (1024px+)
- ✅ Планшет (768px - 1024px)
- ✅ Мобильный (480px - 768px)
- ✅ Маленький мобильный (<480px)
